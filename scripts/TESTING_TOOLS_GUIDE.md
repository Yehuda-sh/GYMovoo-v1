## 🚀 תכונות מתקדמות חדשות

### 📄 **דיווחים אוטומטיים**:

- `accessibility-report-[timestamp].json` - דיווח נגישות מפורט
- `navigation-report-[timestamp].json` - מפת ניווט מלאה
- שמירה אוטומטית עם timestamp לכל בדיקה

### 📖 **איך לקרוא את דיווחי ה-JSON**:

#### **accessibility-report-\*.json**:

```json
{
  "timestamp": "2025-08-03T...",
  "summary": { "totalIssues": 191 },
  "details": {
    "accessibility": { "issues": [...], "stats": {...} },
    "colors": { "issues": [...], "stats": {...} },
    "touchTargets": { "issues": [...], "stats": {...} },
    "accessibleText": { "issues": [...], "stats": {...} }
  }
}
```

#### **navigation-report-\*.json**:

```json
{
  "timestamp": "2025-08-03T...",
  "summary": { "totalIssues": 5 },
  "details": {
    "routes": ["Welcome", "Login", ...],
    "screenComponents": ["Welcome", "Login", ...],
    "navigationCalls": ["Register", "Terms", ...],
    "issues": [{"type": "missing_screen", "route": "Profile", ...}]
  }
}
```

**💡 טיפ**: השתמש בכלים כמו `jq` או עורך JSON לניתוח הדיווחים
