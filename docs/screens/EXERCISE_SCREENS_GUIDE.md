# מסכי תרגילים (Exercise Screens) 💪

## סקירה כללית

מסכי התרגילים מספקים גישה למאגר התרגילים הענק, חיפוש מתקדם, פרטים מלאים על כל תרגיל, ומערכת לימוד אינטראקטיבית.

## 📱 מסכים במודול

### 1. ExerciseListScreen (מסך רשימת תרגילים)

**מיקום:** `src/screens/exercise/ExerciseListScreen.tsx`

**תפקיד:**

- הצגת כל התרגילים במאגר (600+)
- חיפוש וסינון מתקדם
- מעבר מהיר לפרטי תרגיל

**תכונות עיקריות:**

- 🔍 **חיפוש טקסט:** לפי שם תרגיל או שריר
- 🎯 **סינון לפי קבוצת שרירים:** חזה, גב, רגליים וכו'
- 🏋️ **סינון לפי ציוד:** משקולות, מכונות, משקל גוף
- ⭐ **סינון לפי רמת קושי:** מתחיל, בינוני, מתקדם
- 📊 **מיון:** א-ב, פופולריות, קושי, זמן אחרון
- ❤️ **מועדפים:** שמירת תרגילים אהובים

**מבנה התצוגה:**

```
┌─────────────────────────┐
│     Search Bar + 🔍     │
├─────────────────────────┤
│    Filter Chips Row     │
│  [חזה] [גב] [רגליים]    │
├─────────────────────────┤
│     Exercise Cards      │
│  ┌─────────────────┐    │
│  │ 🏋️ Bench Press  │    │
│  │ חזה, כתפיים      │    │
│  │ ⭐⭐⭐⭐⭐        │    │
│  └─────────────────┘    │
└─────────────────────────┘
```

**אפקטי UX:**

- טעינה lazy לביצועים
- אנימציות כניסה חלקות
- Pull-to-refresh עדכון
- Infinite scroll לכמות גדולה

**מיקום:** `src/screens/exercise/ExerciseDetailsModal.tsx`

**תכונות עיקריות:**

- 📝 **הוראות מפורטות:** צעד אחר צעד בעברית ואנגלית
- 🎯 **שרירים מעורבים:** ראשיים ומשניים
- ⚠️ **שגיאות נפוצות:** מה לא לעשות
- 💡 **טיפים מקצועיים:** שיפור ביצוע
- 🔄 **וריאציות:** תרגילים דומים/חלופיים
- 📊 **מטא-דטה:** רמת קושי, זמן, ציוד נדרש

**מבנה המודל:**

```
┌─────────────────────────┐
│    Exercise Image       │
├─────────────────────────┤

│   Muscle Groups Tags    │
├─────────────────────────┤
```

**אינטראקציות:**

- Swipe left/right למעבר בין תרגילים
- Pinch to zoom על תמונות

## 🗃️ מאגר התרגילים

### מבנה הנתונים:

```typescript
interface Exercise {
  id: string;
  name: {
    he: string; // שם עברי
    en: string; // שם אנגלי
  };
  primaryMuscles: string[]; // שרירים ראשיים
  secondaryMuscles: string[]; // שרירים משניים
  equipment: string[]; // ציוד נדרש
  difficulty: "beginner" | "intermediate" | "advanced";
  instructions: {
    he: string[]; // הוראות עברית
    en: string[]; // הוראות אנגלית
  };
  tips: string[]; // טיפים
  commonErrors: string[]; // שגיאות נפוצות
  variations: string[]; // תרגילים דומים
  image?: string; // נתיב תמונה
  gif?: string; // נתיב GIF
}
```

### קטגוריות תרגילים:

- **חזה (Chest):** 45+ תרגילים
- **גב (Back):** 60+ תרגילים
- **רגליים (Legs):** 80+ תרגילים
- **כתפיים (Shoulders):** 40+ תרגילים
- **זרועות (Arms):** 70+ תרגילים
- **ליבה (Core):** 50+ תרגילים
- **קרדיו (Cardio):** 30+ תרגילים

### סוגי ציוד:

- **משקל גוף:** 150+ תרגילים
- **משקולות חופשיות:** 200+ תרגילים
- **מכונות:** 100+ תרגילים
- **ציוד בית:** 80+ תרגילים
- **אביזרי עזר:** 70+ תרגילים

## 🎨 עיצוב ו-UX

### כללי עיצוב:

- **RTL מלא** עם עיצוב שמאל-ימין
- **Typography:** גופנים ברורים לקריאת הוראות
- **Color Coding:** צבע לכל קבוצת שרירים
- **Spacing:** מרווחים נוחים לקריאה ארוכה
- **Imagery:** תמונות ברורות וGIFים איכותיים

### אלמנטי UX:

- **Quick Actions:** הוספה מהירה לאימון
- **Search History:** זכירת חיפושים אחרונים
- **Smart Suggestions:** המלצות בהתאם להיסטוריה

## 🔧 טכנולוגיות

### Performance:

- **FlatList וירטואלי:** לטעינה מהירה של רשימות גדולות
- **Image Caching:** תמונות נטענות פעם אחת
- **Search Optimization:** אלגוריתם חיפוש מהיר
- **Memory Management:** ניקוי זיכרון אוטומטי

### State Management:

- **ExerciseStore:** מאגר התרגילים המקומי
- **SearchStore:** היסטוריית חיפושים
- **FavoritesStore:** תרגילים מועדפים
- **FilterStore:** העדפות סינון
